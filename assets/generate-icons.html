FitGirl Watchlist Icon Generator

This page will help you generate the required PNG icons for the extension.

Instructions:

  1. Open the SVG file in a browser or image editor
  2. Export/save as PNG in the following sizes:
    • 16x16 pixels → save as icon16.png
    • 48x48 pixels → save as icon48.png
    • 128x128 pixels → save as icon128.png
  3. Place all PNG files in the icons/ directory

Alternative: Use online converter

You can use online SVG to PNG converters like:

  • Convertio
  • CloudConvert
  • FreeConvert
