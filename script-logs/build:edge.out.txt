📋 Installation Instructions
============================

🟦 Microsoft Edge:
  1. Open Edge and go to edge://extensions/
  2. Enable "Developer mode" (toggle in left sidebar)
  3. Click "Load unpacked" and select the dist/edge/ folder
  4. Or install the .zip file from dist/

💡 Tips:
  - Make sure to generate PNG icons before building
  - Test the extension on fitgirl-repacks.site pages
  - Run 'pnpm run installers' to create user-friendly installers
  - Check the browser console for any errors
